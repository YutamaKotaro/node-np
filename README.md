# NetProtection API for Node.js

## Develop

```
$ git clone git@github.com:air-closet/node-np.git && cd node-np
$ npm i

--- development ---

$ npm run lint
$ npm run build
$ npm run mocha

or

$ npm test
```

## Scripts

### lint

* eslint

### build

* babel

### mocha

* mocha
* power-assert

### test

This task executes all npm-scripts.
