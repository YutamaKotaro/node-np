# NetProtection API for Node.js

## develop

```
$ git clone git@github.com:air-closet/node-np.git && cd node-np
$ npm i

--- development ---

$ npm run lint
$ npm run build
$ npm run test

or

$ npm run all
```

## npm-scripts

### lint

* eslint

### build

* babel

### test

* mocha
* power-assert

### all

This task executes all npm-scripts.
